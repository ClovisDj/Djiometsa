Game Connect Four.

www.playconnect4.com

Austin Tx.

Year 2017.

Interactive Web application game inspire from the legendary connect four table game. With clean and intuitive UI, the front end side of this apllication is optimize for mobile devices and cross-browser compatible. With the use Bootstrap.

On the back end side, the application is connectect to MySQL for recording the registered players game's scores which are automatically saved and retrived via Ajax GET, POST method. This application also features a CRUD methodology to commit in database and modify user's basic personal infos. Once an user is registered, a welcome email will be delivered to the provided email by admin@playconnect4.com via the AWS SES service. Feel free to access the account: jane, Password: JaneDoe to explore the entire functionality of this web app.

In the core, the game algorithm is implemented in vanilla JavaScript with Jquery for the DOM manipulation. This alorithm uses a probabilistic approach to determine the various game's levels.

Technologies Used: Python, OOP, Django 1.11, MySQL, JavaScript, Django REST framework, Django-mySQL, Bootstrap, Html, Css, Json, Ajax, Jquery, Git, Github, AWS Eb, AWS Ec2, AWS SeS.
