Personal Website.

www.dnclovis.com

Austin Tx.

Year 2017.

This personal presentation website is heavily front end oriented. Making use of a HTML5 Canvas animation on the landing page powered by a JavaScript algorithm. It also make an extenive use of Bootstrap for mobile friendly devices. This wesite features a google maps integration that expands on mouse hover realized with Jquery and javascript.

On the back end, this application is connected to MySQL to take care of the messages sent via the Email Me button which appears on the bottom right of the pages. This last, implemented with Ajax.

Technologies Used: Python, Django 1.11, MySQL, JavaScript, Django-mySQL, Bootstrap, Html, Canvas Animantion, Ajax, Jquery, Git, Github, AWS Eb, AWS Ec2, AWS SeS.

Github Repository: github.com/ClovisDj/dnclovis
